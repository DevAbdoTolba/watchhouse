3. `class FrameQueue`:
   - Constructor: `__init__(self, maxlen: int = 2)` — stores `self._dq = deque(maxlen=maxlen)`, `self._lock = threading.Lock()`, `self.drop_count = 0`, `self.push_count = 0`.
   - `put(self, item: Tuple[np.ndarray, float]) -> None`: under lock, if `len(self._dq) == self._dq.maxlen` then increment `self.drop_count` (the append about to happen will evict the oldest); then `self._dq.append(item)`; `self.push_count += 1`.
   - `get_latest(self) -> Optional[Tuple[np.ndarray, float]]`: under lock, return `self._dq[-1]` if non-empty else `None`. (Phase 2's InferenceWorker will use this; we test it here.)
   - `pop_oldest(self) -> Optional[Tuple[np.ndarray, float]]`: under lock, `self._dq.popleft()` if non-empty else `None`.
